Array Methods()

forEach()

Open browser console to view the output

This method is used to iterate each elemnts in an array. For example

forEach method modifies the original array 

            var arr1 = [1, 2, 3, 4, 5];
            arr1.forEach((x) => {
                console.log(x);
            })

Output

  • 1
  • 2
  • 3
  • 4
  • 5
map()

This methods loops through each element & returns a new array. Hence do not modify the original array For example 

            var arr1 = [1, 2, 3, 4, 5];
            var mapArr = arr1.map((x) => x * 2);
            console.log(mapArr);

Output

  • [2, 4, 6, 8, 10]
filter()

This method is also iterates on every array element & returns an new array as per the conditions written down For example 

            var arr1 = [1, 2, 3, 4, 5];
            var filterArr = arr1.filter((x) => x === 2 || x === 4);
            console.log(filterArr);

Output

  • [2, 4]
reduce()

This method returns only one element after the given condition is satisfied

reduce() method takes 2 arguments

  • acc - it's just like a variable that is used to store & return a final value
  • curr - the current elemnt of an array where iteration is happening
For example 

            var arr1 = [1, 2, 3, 4, 5];
            var sumReduce = arr1.reduce((acc, curr) => {
                return acc + curr;
            });
            console.log(sumReduce);

Output

  • 15
